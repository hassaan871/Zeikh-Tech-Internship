API Endpoints:

Register User: http://localhost:PORT/api/register-user
Login User: http://localhost:PORT/api/login-user
Add Task: http://localhost:PORT/api/add-task
Search Task by heading: http://localhost:PORT/api/search-by-heading
delete Task: http://localhost:PORT/api/delete-task/:id
get all Tasks: http://localhost:PORT/api/get-all-task/
get completed Tasks: http://localhost:PORT/api/get-completed-task/
get incompleted Tasks: http://localhost:PORT/api/get-incompleted-task/
